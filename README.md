##  Как выгрузить на Vercel

### Способ 1: Через Vercel CLI (Рекомендуется)

#### Шаг 1: Установите Node.js
Скачайте и установите с https://nodejs.org/

#### Шаг 2: Установите Vercel CLI
Откройте командную строку и выполните:
```bash
npm install -g vercel
```

#### Шаг 3: Войдите в аккаунт Vercel
```bash
vercel login
```
Следуйте инструкциям в консоли (можно войти через GitHub, GitLab или email).

#### Шаг 4: Разверните сайт
В папке с проектом выполните:
```bash
vercel
```

**Первый запуск:**
- Нажмите `Y` для продолжения
- Введите имя проекта (или оставьте по умолчанию)
- Выберите папку `./` (текущую)
- Нажмите `Y` для деплоя

**Последующие запуски:**
```bash
vercel --prod
```

### Способ 2: Через GitHub + Vercel (Автоматический деплой)

#### Шаг 1: Загрузите проект на GitHub
```bash
# Инициализируйте git репозиторий
git init

# Добавьте все файлы
git add .

# Сделайте первый коммит
git commit -m "Initial commit - i-CAR website"

# Добавьте удалённый репозиторий (замените YOUR_USERNAME на ваш логин)
git remote add origin https://github.com/YOUR_USERNAME/i-car.git

# Отправьте на GitHub
git push -u origin main
```

#### Шаг 2: Подключите Vercel
1. Зайдите на https://vercel.com
2. Войдите через GitHub
3. Нажмите **"Add New Project"**
4. Выберите репозиторий `i-car`
5. Нажмите **"Deploy"**

**Готово!** Vercel автоматически задеплоит сайт и выдаст URL.

#### Шаг 3: Автоматические обновления
Теперь при каждом `git push` сайт будет обновляться автоматически!

```bash
# Внесите изменения
git add .
git commit -m "Updated styles"
git push
```

Через 1-2 минуты изменения будут на сайте.

## 🌐 Доменное имя

### Бесплатный домен Vercel
Ваш сайт будет доступен по адресу:
```
https://i-car.vercel.app
```

### Свой домен (например, i-car.ru)

1. Купите домен (reg.ru, nic.ru, namecheap.com)
2. В панели Vercel:
   - Project Settings → Domains
   - Добавьте ваш домен
3. В настройках домена пропишите:
   ```
   Тип: A
   Имя: @
   Значение: 76.76.21.21
   ```

## 📊 Analytics

Добавьте Vercel Analytics для отслеживания посещений:

1. В панели Vercel: Analytics → Enable
2. Добавьте скрипт в `<head>` всех страниц:

```html
<script src="https://vercel.live/script.js" data-analytics="true"></script>
```
